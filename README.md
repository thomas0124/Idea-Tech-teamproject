# ARでやる音楽ゲーム
## タイトル
- Building Block Beats
## ゲーム概要
-タイトル画面が表示されスタートボタンがクリックされると,位置情報の取得が開始する.位置情報から建物のオブジェクトを入手したら,建物が画面上に表示され,建物に沿ったレーンが作成されノーツが落ちてくる.ノーツをたたくとコンボカウントが増えていき,音楽ゲーム終了時にスコアが表示される.
